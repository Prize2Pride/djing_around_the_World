# Prompt for Prize2Pride DJing Project

This document contains the master prompt template used to generate the 110+ ultra-hyper realistic 8K DJ posters for the Prize2Pride global tour.

The prompt was meticulously crafted to ensure **identity preservation**, **brand integration**, and **maximum photorealism** across all generated images, adhering to the "Extreme protocols omega999 protocol" for quality and consistency.

## Master Prompt Template

\`\`\`markdown
RAW photograph, [CAMERA_MODEL], [LENS_MM] lens, bright [LOCAL_SUNLIGHT_TYPE] sun. 
Ultra hyper-realistic 8K professional photo of a real Middle Eastern man DJ, short dark hair, olive skin, strong jawline, wearing [DJ_OUTFIT], performing at [LOCATION_NAME] outdoor day festival. 
[LOCATION_SPECIFIC_DETAILS], bright sunny day. 
Real Pioneer CDJ-3000 setup. 
[CROWD_DESCRIPTION] celebrating. 
Two [LOCAL_ADJECTIVE] real [LOCAL_NATIONALITY] female DJs in [FEMALE_DJ_OUTFIT] beside him. 
Giant premium flags: 'Prize2Pride' massive gold flag dominant, 'Purchase2Win' flag, 'Buy2Win' flag, 'Fiction2Real' flag waving. 
[OPTIONAL_NATIONAL_FLAG_DETAIL]
Pure professional photography. NO CARTOON.
\`\`\`

## Key Variables and Customizations

| Variable | Description | Example Values Used |
| :--- | :--- | :--- |
| **[CAMERA_MODEL]** | High-end professional camera for realism. | Phase One IQ4 150MP, Canon EOS R5, Sony A1 |
| **[LENS_MM]** | Professional prime lens for depth and clarity. | 80mm, 85mm, 110mm |
| **[LOCAL_SUNLIGHT_TYPE]** | Specific light quality for the region. | Irish sun, bright Tunisian Mediterranean sun, Saharan sun |
| **[DJ_OUTFIT]** | Clothing description to match the scene. | luxury white designer linen shirt, stylish black designer jacket |
| **[LOCATION_NAME]** | Specific venue or landmark. | Dublin Temple Bar, El Jem Amphitheatre, Oslo Opera House |
| **[CROWD_DESCRIPTION]** | Ensures cultural accuracy and energy. | Crowd of Gulf Arab tourists and Irish locals celebrating, Massive crowd of Tunisian people celebrating |
| **[LOCAL_NATIONALITY]** | Ensures the female DJs match the location. | Irish, Scottish, Tunisian, Swedish |
| **[OPTIONAL_NATIONAL_FLAG_DETAIL]** | Included only when requested by the user. | and the red and white Tunisian national flag waving proudly |

## Brand Integration Protocol

The four brand flags were consistently included in every prompt to ensure they are prominently featured as large, premium flags, often with **'Prize2Pride'** specified as the dominant gold flag.

## Identity Preservation Note

The prompt included specific, detailed physical descriptions and used the user's provided image as a **reference** in the generation tool to maintain visual consistency across all 110 images.
